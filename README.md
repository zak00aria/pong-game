# Classic Pong Game

A classic pong game written in C.

![Project screenshot on wokwi.com](./wokwi_screenshot.png)

The game is written in C (and it uses esp-idf), we used FreeRTOS to manage tasks.
- for the hardware we used an ESP32 board (using esp-idf), 
- for display we used an ssd1306 OLED (controlled over i2c),
- and a Buzzer for sound outputs (controlled using PWM).

You can check project simulation on wokwi from [here](https://wokwi.com/projects/462722583293411329)
